## FAQ

**Q: The matrix is not working properly, why?**

A: First thing, check the **MEMORY** of your ESP32, to operate the LED matrix need lots of memory compare to operate other modules.
[Check official document here.](https://github.com/mrcodetastic/ESP32-HUB75-MatrixPanel-DMA#required-memory)
